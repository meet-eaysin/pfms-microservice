# Kong API Gateway Configuration Templates

This directory contains reusable configuration templates and documentation

## Available Templates

### Service Template

Use this template when adding a new microservice to the gateway.

### Route Template

Use this template when adding new routes for a service.

### Plugin Templates

Common plugin configurations that can be reused across services.

## Usage

Copy the relevant template and customize it for your service. Templates use placeholders like `{SERVICE_NAME}` that should be replaced with actual values.
